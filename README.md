# Henry Rado-Lightfoot Website

This is a small static website for GitHub Pages. It has no build step and no dependencies.

## Files

- `index.html` controls the page content.
- `styles.css` controls the colors, layout, and responsive design.
- `script.js` draws the animated background.
- `CNAME` tells GitHub Pages to use `henryradolightfoot.com`.
- `.nojekyll` tells GitHub Pages to publish the files directly.

## First Change

1. Open `index.html`.
2. Change one sentence in the Projects section.
3. Commit the change.
4. Wait for GitHub Pages to publish.
5. Refresh the live site.

## Public Safety Rules

- Do not add school names, exact locations, daily schedules, phone numbers, private photos, passwords, or payment details.
- Ask a parent before adding photos, contact forms, sign-up forms, downloads, or links to external accounts.
- Keep this site simple and project-focused.
